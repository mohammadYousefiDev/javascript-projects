We want to increase or decrease a number
1. Create number box to display default number(zero is default)
2. Create an increase, reset and decrease button
3. Create base increase method
4. Get current number of box
5. Increase one unit to it
6. check number, if its positive, set green color and if its zero, set dark color
7. Create base reset method
8. Get current number of box
9. set number box value to zero
10. set number color to dark
11. Create base decrease method
12. Get current number of box
13. decrease one unit to it
14. check number, if its negative, set red color and if its zero, set dark color
